# PureMuga

PureMuga is a static website built using HTML and CSS to showcase authentic Assamese silk garments and traditional attire.

The website is designed as a visual showcase and enquiry platform for a physical store. Customers are expected to visit the store for purchases.

## Project Status
- Home page design completed
- About page completed
- Products page structure ready (images will be added later)
- Contact page completed
- Codebase cleaned with minimal issues remaining

## Pages
- Home
- Products
- About
- Contact

## Features
- Clean and elegant UI inspired by Assamese heritage
- Organized image folder structure
- Consistent header and footer across pages
- Responsive layout
- Enquiry-focused (no online ordering)

## Tech Stack
- HTML5
- CSS3
